# 笺云科技 × 时光绿径待办 — 企业介绍落地页设计

## 概述

ICP 备案审核用企业介绍页，以产品展示为主、企业资质为辅。单页滚动式 HTML，绿色主题 + 毛玻璃 + 超圆角设计语言，与小程序 DESIGN.md 保持一致。纯静态，无构建步骤，部署于 `intro.yzjtiantian.cn`。

## 页面结构

### 1. Hero 区
- 全屏绿色渐变背景（`135deg, #00b26a → #3ddaa0`）
- 毛玻璃导航条（`backdrop-filter: blur(20rpx)`）
- 品牌名「笺云科技工作室」/ **CloudNote Studio**
- 标语
- 小程序名「时光绿径待办」+ 一句话介绍
- 小程序码（二维码图片占位）

### 2. 产品理念
- "时光绿径" 的品牌寓意：成长、自然、进步
- 设计哲学：绿色即情绪，毛玻璃与超圆角视觉语言
- 灵感源于「在时光中留下一串绿色足迹」

### 3. 功能矩阵
卡片网格展示核心功能模块：
- **待办管理** — 优先级、标签、拖拽排序
- **组合系统** — 文件夹归类、共享协作、成员分配
- **日历集成** — 截止日期标记、日视图
- **语音输入** — 微信语音识别快速创建
- **数据洞察** — 完成率、趋势图表、每日分析
- **离线同步** — 离线优先、增量同步、冲突合并

### 4. 设计亮点
- 毛玻璃导航 + 超圆角卡片（32rpx）
- 微动效反馈（完成脉冲、星标弹跳）
- 单色绿色视觉系统，一致且克制

## 设计语言（继承 DESIGN.md）

- **主色**: `#00b26a`
- **辅色**: `#3ddaa0`（渐变高光），`#e3f5eb`（浅底）
- **圆角**: 32rpx → 换算为 CSS `clamp(16px, 4.267vw, 24px)` 等
- **毛玻璃**: `rgba(255,255,255,0.78)` + `backdrop-filter: blur(20rpx)`
- **卡片阴影**: `0 8rpx 40rpx rgba(0,0,0,0.1)`
- **字体**: 系统原生栈（PingFang SC / Microsoft YaHei）
