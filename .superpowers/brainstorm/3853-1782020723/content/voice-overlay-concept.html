动画时序设计

扩散（按下）

  1. 获取 FAB 按钮中心位置
  2. clip-path 圆形从 0 → 200% 展开
  3. cubic-bezier(0.34, 1.56, 0.64, 1) 弹性
  4. 同时开始麦克风录音

收缩（松手）

  1. clip-path 圆形从 200% → 0 收回
  2. cubic-bezier(0.55, 0.085, 0.68, 0.53) 缓出
  3. 动画完成后跳转 add-todo
  4. 携带识别到的语音文本
